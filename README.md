# SAHYOG - HIV Awareness Platform

A multi-language, interactive platform dedicated to HIV/AIDS awareness, education, and support.

## Features

- **Multi-language Support**: English, Hindi, Malayalam, Tamil, Kannada, Telugu.
- **Awareness Hub**: Comprehensive information about HIV transmission, prevention, testing, and treatment.
- **Interactive Games**: Gamified learning experiences to test knowledge and break myths.
- **Stories of Hope**: A safe space for sharing and reading anonymous stories from the community.
- **Modern Design**: Clean, accessible, and responsive interface with a focus on user experience.

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, Framer Motion
- **State Management**: Zustand
- **Internationalization**: i18next
- **Routing**: React Router DOM

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Project Structure

- `src/pages`: Main application pages (Homepage, AwarenessHub, Games, Stories, etc.)
- `src/components`: Reusable UI components
- `src/locales`: Translation files
- `src/store`: Global state management
- `src/services`: Utility services (translation, etc.)

## Design System

- **Primary Color**: Rose (`#e11d48`) - Symbolizing care and urgency.
- **Secondary Color**: Gold (`#f59e0b`) - Symbolizing hope and value.
- **Neutral**: Slate (`#1e293b`) - For text and structure.
- **Typography**: Sans-serif (Inter/Noto Sans) for readability across languages.
